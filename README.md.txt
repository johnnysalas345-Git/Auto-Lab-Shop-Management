# Auto Lab Shop Management

Garage work order and customer management system for Auto Lab Ltd.

## Quick Start

1. Install dependencies: `npm install`
2. Copy `.env.local.example` to `.env.local`
3. Add Supabase keys to `.env.local`
4. Run: `npm run dev`

## Technologies

- React 18
- Vite
- Supabase
- Lucide Icons

## Database

Uses Supabase project: `gvsbbpuuoxluqaiovtvx`

Tables: customers, vehicles, work_orders, inventory, inspections
